# TMS-EEG-Microstates

This MATLAB toolbox implements a comprehensive analysis pipeline for investigating how transcranial magnetic stimulation (TMS) affects EEG microstate dynamics. The code analyzes microstate occurrence patterns (using compositional data analysis with centered log-ratio transformation) and transition probabilities, employing Threshold-Free Cluster Enhancement (TFCE) for robust statistical testing. To run the analysis, simply set the parent_dir path in run_analysis.m to your data directory, ensure your data files follow the naming convention [Dataset]_[Visit].mat, and execute run_analysis.m. The script will automatically extract microstate frequencies, perform statistical comparisons across stimulation sites (left/right DLPFC and left M1), and generate publication-ready figures showing temporal dynamics with 95% confidence intervals and significance markers. Results, including TFCE statistics and effect sizes, are saved to the results/ folder.
